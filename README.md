Landing Page - CTTOEB

Landing page institucional desenvolvida para o **CTTOEB**, com foco em **comunicação clara, organização de informações e fortalecimento da presença digital**.

Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de criar uma página moderna e acessível para apresentar a instituição, seus objetivos e serviços de forma clara e estruturada.

A landing page foi pensada para facilitar o acesso à informação e melhorar a comunicação com o público.

---

Funcionalidades

Apresentação institucional clara  
Organização de conteúdo por seções  
Navegação simples e intuitiva  
Layout responsivo para diferentes dispositivos  
Estrutura visual limpa e profissional  

Objetivo

Criar uma página institucional capaz de:

- Apresentar a organização de forma profissional  
- Melhorar a comunicação com o público  
- Facilitar o acesso às informações  
- Fortalecer a presença digital da instituição  

Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript 

Destaques do Projeto

- Estrutura semântica para melhor organização do conteúdo  
- Interface intuitiva e de fácil navegação  
- Design alinhado ao perfil institucional  
- Foco em experiência do usuário (UX)
